* Kosmo_LED_Hole
* Kosmo_Panel_Dual_Mounting_Holes

* Kosmo_Panel_Mounting_Hole
* Kosmo_Panel_Slotted_Mounting_Hole
* Kosmo_Pot_Hole
* Kosmo_Slide_pot_Alpha_60mm
* Kosmo_Switch_Hole
* Kosmo_Trimmer_Pot_Hole
* Little_Jack_Hole
* OLED_128x64
* Push_Button_A-3487
